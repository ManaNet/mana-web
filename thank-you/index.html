Frequently Asked Questions

Are we allowed a refund?

As much as we wish to allow refunds, all purchases are directly sent to our service providers and other providers to give us some leeway for cases when we somehow forget to pay for a service or a host provider. This allows the bot to be active almost all the time with a promised 99.9 percent uptime with nearly three hundred thousand images a month sent.

How do we generate server-licenses?

You can generate server licenses by using the command pa.license which would then generate a license for you to redeem on the server of choice through the command pa.redeem [license], please note that pa. is the default prefix of Mana which is changeable, use the prefix the server uses. To find it, simply ping Mana on the server with no context.

How do we change the bot’s timing on Yuriverse?

To change the timing from hourly to quarterly images on Yuriverse, simply use the command: pa.yuriverse speed which would then switch it to quarterly if it is on hourly and hourly if it is on quarterly.
